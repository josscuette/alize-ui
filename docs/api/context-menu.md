ContextMenu component - A context menu (right-click menu)

Provides an accessible context menu that appears on right-click.
Built on Radix UI primitives for accessibility.

#### Param

ContextMenu props including open, onOpenChange, and standard Radix UI ContextMenu attributes

#### Returns

A ContextMenu component

#### Example

```tsx
<ContextMenu>
  <ContextMenuTrigger>Right-click me</ContextMenuTrigger>
  <ContextMenuContent>
    <ContextMenuItem>Item 1</ContextMenuItem>
  </ContextMenuContent>
</ContextMenu>
```
